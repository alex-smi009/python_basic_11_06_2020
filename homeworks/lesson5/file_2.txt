В первом примере выполняется открытие файла my_file.txt в режиме только чтения, который является режимом по умолчанию (можно не указывать). При этом указано только имя файла с расширением. Полный путь к файлу не указан. Интерпретатор Python будет искать данный файл в директории, где расположен файл-модуль, в котором мы выполняем открытие файла my_file.txt.
